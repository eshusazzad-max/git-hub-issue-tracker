1----What is the difference between var, let, and const?

JavaScript এ var, let,এবং const তিনটাই variable declare করার জন্য ব্যবহার করা হয় কিন্তু এদের behavior আলাদা

var হচ্ছে পুরোনো পদ্ধতি এটা function scoped এবং একই variable আবার declare করা যায়

let block scoped এবং সাধারণত তখন ব্যবহার করা হয় যখন variable এর value পরে change হতে পারে
const ও block scoped কিন্তু একবার declare করলে তার value আর change করা যায় না



2--- What is the spread operator (...)?

Spread operator(...)ব্যবহার করা হয় array বা object এর element গুলোকে expand বা copy করার জন্য

এটা ব্যবহার করলে খুব সহজে array merge বা copy করা যায়।




3--- What is the difference between map(), filter(), and forEach()?

এগুলো JavaScript array method

map(): প্রতিটা element এর উপর operation করে নতুন array return করে

filter(): condition অনুযায়ী কিছু element নিয়ে নতুন array return করে

forEach(): শুধু loop চালায় কিন্তু নতুন array return করে না



4--- What is an arrow function?

Arrow function হচ্ছে JavaScript এ function লেখার একটি ছোট এবং modern syntax

এটা => symbol ব্যবহার করে লেখা হয় এবং code অনেক cleaner হয়



5--- What are template literals?

Template literals ব্যবহার করা হয় string এর মধ্যে variable সহজভাবে insert করার জন্য
